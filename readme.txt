Files list

aia-template14.ltx      latex source file - use this for your document
aia-template14.pdf      compiled result of the template file
aia-blocks.pdf          blocks image used in the template
aia-refs.bib            example bib file for template references
cardiff.bst             Cardiff university bibtex style for Harvard
files                   folder of files needed by the template

Environment variable TEXINPUTS can be changed to point to the 'files'
folder or OR copy the contents to the same folder as
aia-template14.ltx so that the LaTeX compiler (pdflatex) can locate
these files.

Updates and/or missing files can be found in CTAN (see Dr Ian Wood's
intranet site Modules > AIA area).
